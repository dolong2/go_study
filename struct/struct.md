# Go lang 구조체
go lang에서 구조체는 필드값만 가질수 있으며 메서드는 가질 수 없다.

메서드는 별도로 분리해서 사용

```
type 구조체명 struct {
    var 필드명 타입
}
```
```
var 변수명: 구조체명 = 구조체명{필드값}
var 변수명: 구조체명 = 구조체명{필드명: 필드값}
```

구조체가 사용되기전에 필드가 초기화 되어야할때 사용하기 위한 메서드를 만들어서 사용할 수 있음

포인터를 전달해야함
```
func newDict() *구조체명 {
    변수명 := 구조체명{}
    //원하는 필드 초기화
    return &변수명
}
```
